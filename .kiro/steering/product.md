# Product Overview

## AI Ghost Story Generator

A web application that generates horror stories using AI based on user prompts. Users enter a short prompt (3-200 characters) and receive a chilling, atmospheric ghost story generated by OpenAI's GPT-3.5-turbo.

### Key Features

- Real-time story generation with OpenAI integration
- User authentication via Clerk
- Responsive horror-themed UI with custom animations
- Error handling with retry functionality
- Loading states with timeout warnings (15s warning, 25s timeout)
- Story length: 200-800 words

### User Flow

1. User enters a horror prompt (e.g., "abandoned house", "witch forest")
2. System validates prompt length (3-200 characters)
3. Story generation begins with loading animation
4. After 15 seconds, timeout warning appears
5. Story displays with option to generate new story
6. Errors show with retry or start over options
